TAGS AND ELEMENTS
HTML

The root element.

DOCTYPE

It is used to define the document type.

HEAD

It is the tag which does not contain any text but encompasses all the other tags.

TITLE

This tag is used when you want to give a title of your content.

THEAD

used to group the header content of the table in a file. It represents the top row of table which provides the heading for the columns.

TR

This tag is passed inside the table tag it used to declare the table row.

TD

This tag is passed inside the <tr> tag.It can contain any content inside itself.

BR

It is the line break command which is used to break the lines inside the file.

BODY

Anything which you see on the website is written under the tag.

DIV

The division tag, is used to define the division of the content in the file.

SPAN

It is used to wrap the entire text in the file. It is basically used to group the inline content of the file.

TBODY

It is another one of the table tags which is used to group the header content of the table in a file.

SCRIPT

When you want to define the client-side script inside your file, you use script tag

IMG

If you wish to put any image in the HTML files, you can do the same using img tag.

UL

used to define the unordered list in the file. It posts the limits with bullets on the website.

U

used along with the tag to pass the hierarchy of list in the file. When you want to pass a list inside a list we make use of li tag
